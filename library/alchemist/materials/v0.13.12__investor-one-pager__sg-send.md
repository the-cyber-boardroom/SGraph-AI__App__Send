## The Problem

Confidential files travel through channels where the server can read everything. Email is plaintext. Slack is searchable by admins. WeTransfer has no encryption. Enterprise data rooms (Intralinks, Datasite) charge £3-10K/month and hold the encryption keys themselves.

**No mainstream tool offers: end-to-end encryption + no app for the recipient + verifiable + self-hostable + multi-platform.**

---

## The Solution

**SG/Send** — zero-knowledge encrypted file transfer. Encrypt in the browser, decrypt in the browser. The server never sees your files.

1. Sender drops a file in the browser (or desktop app, or mobile app)
2. File is encrypted (AES-256-GCM) before upload
3. Server stores only encrypted bytes — **never sees the plaintext**
4. Recipient opens a link, decrypts in their browser
5. **No app. No account. No install.**

> "Foolproof — like Google search" — Senior security professional, 25 years experience
