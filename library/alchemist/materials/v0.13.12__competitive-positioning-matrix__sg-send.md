## 3. Unique Differentiators (No Competitor Has These)

### 3.1 AI Workspace with Zero-Knowledge Preserved

SG/Send's workspace integrates LLMs (OpenRouter cloud + Ollama local) while maintaining zero-knowledge architecture. Documents load from encrypted vault, transform via LLM in the browser, save back encrypted. **The server never sees the document content OR the LLM conversation.**

Every competitor that adds "AI features" routes documents through their server AND the LLM provider. SG/Send keeps everything client-side — two fewer parties see your data.

**No competitor offers this.**

### 3.2 Multi-Platform from One Codebase

Web + Desktop (Tauri, Apple-signed) + Mobile (Android, real device tested) + CLI — all from one vanilla JS codebase using Web Components. No framework dependencies. IFD (Iterative Feature Development) methodology means every platform gets the same features with minimal branching.

**No competitor achieves this convergence from a single codebase.**

### 3.3 Seven Deployment Targets

Same application runs on: Lambda, Docker, GCP Cloud Run, Fargate, EC2/AMI, PyPI, CLI. Enterprises can deploy in their own infrastructure, their own jurisdiction, air-gapped if needed.

**No competitor offers this deployment flexibility.**

### 3.4 Agentic Team Velocity

1 human + 18 AI agent roles. Feature request to production in 90 minutes. Desktop app shipped in 1 day. 283 commits in 9 days. Monthly burn in the low thousands.

**No competitor has this operational model.**

## 5. The Competitive Defence

### Structural Moats (Not Replicable by Feature Addition)

1. **Zero-knowledge architecture** — requires clean-slate engineering. WeTransfer, MASV, and Intralinks cannot add this incrementally.
2. **Browser-native PKI** — no external dependencies. Tresorit requires enterprise PKI infrastructure.
3. **Encryption trilemma resolution** — Secure Pod architecture provides zero-knowledge + multi-user access + revocable permissions simultaneously.
4. **Open source** — "anyone can verify" is the strongest security claim possible. Tresorit, MASV, Intralinks cannot make this claim.

### Velocity Moats (Operational Advantage)

5. **Agentic team model** — 1 human + 18 AI agents. Feature velocity that a 50-person team would struggle to match.
6. **IFD methodology** — ship in thin slices, measure, iterate. New features reach all platforms simultaneously.
7. **Zero-framework frontend** — vanilla JS Web Components mean zero supply chain risk, zero dependency conflicts, zero build step delays.

### Network Moats (Compound Over Time)

8. **PKI registry** — more published keys = more valuable network
9. **Component library** (tools.sgraph.ai) — more products depend on it = harder to compete
10. **Multilingual at scale** — 17 locales from day one, growing. Competitors starting now can't catch up in non-English markets quickly.
