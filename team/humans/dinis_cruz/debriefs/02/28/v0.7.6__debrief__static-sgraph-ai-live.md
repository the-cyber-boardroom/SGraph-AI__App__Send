## Executive Summary

The SGraph AI marketing website went live on its own CDN infrastructure at **static.sgraph.ai**. This is the first time the project has deployed a public-facing website through its own S3 + CloudFront pipeline. The site is live, the CI pipeline works, and every release is immutably versioned using IFD methodology.

## Why This Matters

- **First public CDN deployment** --- sgraph.ai now has its own infrastructure for static content, independent of any third-party hosting
- **Repeatable pipeline** --- Any push to the website directory auto-deploys via CI with zero manual steps
- **IFD versioning** --- Every release is immutable and independently addressable (`v0/v0.7/v0.7.7/`), enabling instant rollback and audit
- **Zero external dependencies** --- Self-hosted fonts, relative paths, no CDN dependencies at runtime
- **Foundation for static.sgraph.ai** --- The same bucket and pipeline can serve the Send app UI, documentation, and any future static sites
