# Debrief: MCP Milestone — First AI-to-Human Zero-Knowledge Encrypted Transfer

**Version:** v0.6.20
**Date:** 25 February 2026
**Session:** Explorer — MCP Integration + Milestone Capture

---

## Executive Summary

Today Claude.ai web generated a PDF, encrypted it with AES-256-GCM, uploaded it through MCP tools to SGraph Send, and you decrypted it in your browser. **The server never saw the plaintext.** This is the first demonstrated AI-to-human zero-knowledge encrypted communication through a production system.

The Agentic Security investment thesis (#5 from the 28-brief platform vision) went from theoretical to demonstrable in a single session.

---

## What Was Delivered

### The Working Feature
- MCP server integration on User Lambda (via `fastapi-mcp`)
- Stateless HTTP transport for Claude.ai web connector
- Base64 encoding for binary content over JSON
- Access token authentication for MCP tool calls
- Tool name shortening for Claude.ai's 64-char limit
- **Critical fix:** Raw bytes extraction from MCP JSON wrapper (the fix that made it work)

## Key Decisions

| Decision | Impact |
|----------|--------|

| User Lambda only for MCP | Minimised attack surface per AppSec |
