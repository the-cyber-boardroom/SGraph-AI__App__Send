## Security Findings (AppSec)

Two **critical** issues with the individual file storage proposal:
1. **Plaintext hashes in tree structure** enable known-document attacks — must be removed
2. **Write auth via decryption key is insufficient** — need separate signing key for updates

Both must be resolved in a design document before implementation starts.
