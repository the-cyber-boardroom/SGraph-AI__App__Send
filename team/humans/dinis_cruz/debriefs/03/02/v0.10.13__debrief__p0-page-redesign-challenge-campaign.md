**Date:** 2026-03-02

**Session:** Explorer team — Designer, Sherpa, Ambassador, Architect, Dev

## Executive Summary

Completed a full P0 page redesign of the sgraph.ai website to optimise the purchase funnel for the SG/Send Challenge outreach campaign. The primary goal was reducing the path from homepage to Stripe payment from 4 clicks to 1 click. "Buy Credits — £5" is now the primary CTA on the homepage, linking directly to the Stripe checkout. The Early Access (LaunchList) flow is preserved as a secondary option.

Five role reviews were produced before implementation to evaluate the proposal from design, UX, marketing, architecture, and implementation perspectives. All five approved the changes. Implementation touched 20 HTML files across the `en-gb` locale.

## What Was Delivered

### 1. Homepage Hero CTA Swap
- **Primary CTA:** "Buy Credits — £5 →" (coral button, links directly to Stripe checkout)
- **Secondary CTA:** "Try Free →" (ghost/outline button, links to `/early-access/` keeping LaunchList)
- **Rationale:** Challenge campaign recipients land on homepage — the purchase action must be the most visually prominent element

### 2. Sub-Hero Text Update
- **Before:** "Open source · Zero-knowledge · Encrypted file sharing"
- **After:** "Open source · £0.01/file · 500 transfers"
- **Rationale:** Reinforces the value proposition immediately — price per file and what £5 gets you

### 3. Get Started Section Swap
- **Starter Pack** is now the featured card with a "Recommended" badge and direct Stripe link
- **Early Access Program** is now the secondary card
- **Rationale:** Aligns the mid-page conversion point with the hero CTA — consistent purchase emphasis throughout the page

### 4. Pricing Page Stripe Direct Link
- **Multi-Tenant "Buy Credits"** now links directly to Stripe (previously went to `/pricing/cloud/` intermediate page)
- **Rationale:** Removes one unnecessary click from the pricing→purchase path

### 5. Navigation Reorder
- **Before:** Product, Architecture, Pricing, Team
- **After:** Product, **Pricing**, Architecture, Team
- **Rationale:** Pricing is a higher-priority page for campaign visitors than Architecture. Applied consistently across all 20 en-gb HTML files.
