## Customer Journey (Now Working)

```
Stripe Checkout → Payment Complete → Redirect to /welcome#<id>/<key>
    → Fetch encrypted token → Decrypt in browser → Save to localStorage
    → "You're all set!" (497 uses remaining) → "Start sending files" → Upload page
```

The entire flow is zero-knowledge: the access token is encrypted before upload, and decrypted client-side in the customer's browser. The server never sees the plaintext token during delivery.
