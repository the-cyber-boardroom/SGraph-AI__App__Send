## Key Findings

### 1. JS API is Production-Validated

The 09 Apr debrief (doc 256) confirms: 15-second deterministic infographic generation. Claude writes the Playwright script once, runs it once. A full 6-7 slide deck works end-to-end. The flaky minutes-long browser automation is gone.

**Status of the pipeline today:**
- `tools.sgraph.ai` infographic generator: EXISTS (JS API v0.1.36)
- Playwright boilerplate to call it: EXISTS (session briefing v2)
- PlaybookLM product wrapping it: PROPOSED — does not exist yet
- Python package to automate it: PROPOSED — does not exist yet

### 2. PlaybookLM is Named and Specified

"PlaybookLM" was selected from a 10-candidate evaluation (doc 254). It is a Notebook LM competitor that gives users what Notebook LM hides: the presentation doc, per-slide briefs, prompts, logs, and independent regeneration. Branding is just another source document.

The architecture is sound. The tools team received PDF briefs (docs 262, 263) but these could not be read this session (poppler-utils not available). The Librarian has noted the content from context.

### 3. Pure View Mode is the Immediate Build

The mobile sharing loop is broken today: recipients open a vault link on their phone and see a file browser with a navigation panel. The content is there but the experience is wrong.

Pure view mode (doc 261) fixes this. `?view=pure` in the URL → hide folder chrome → show only `_page.json` content full-width. Estimated 1 session. This is the "last mile" of every delivery workflow we've designed since March.

**This should ship before PlaybookLM.**

### 4. One Code Fix Was Caught

**Commit `1551a2b` (13 April)** — vault markdown images were showing as empty boxes. Root cause: `markdown-parser-v031.js` outputs `<img data-md-src="url">` (no `src`) to defer blob URL loading; browse was only querying `img[src]`. Fixed in `send-browse--v0.3.2.js`. Reality document updated.

### 5. New Product Category: Evidence-Backed News Reports

The Sonar API brief (doc 260) opens a new product direction: current news + citations + infographics = shareable evidence-backed reports. The brief correctly identifies this as exploration-first (understand the citation format before building the pipeline). Perplexity Sonar's real-time data + citation chain is the differentiator vs standard LLMs.

**Prerequisite:** Confirm Sonar is available via OpenRouter before building anything.
