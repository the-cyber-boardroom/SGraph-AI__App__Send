## The Concept

A tool at `send.sgraph.ai/tools/storage-viewer` that lets anyone see exactly what SG/Send stores in their browser. Same pattern as the SSH keygen and PGP keygen — a standalone, transparent utility tool.

This serves three purposes:

| Purpose | Value |
|---|---|
| **Transparency** | Users can see exactly what data SG/Send stores locally. No hidden cookies, no tracking, no mystery. |
| **Debugging** | Developers and admins can inspect keys, vault contents, cached data, and diagnose issues. |
| **Proof of no cookies** | We can publicly demonstrate: "Look — no tracking cookies. No analytics. Here's everything we store, and here's why." This is a marketing differentiator. |

## Key Features

### View and Edit

Every stored item can be viewed and (where appropriate) edited. This is a power-user feature for debugging — not for everyday use, but invaluable when things go wrong.

### Export as JSON

Export everything SG/Send stores locally as a JSON file. Useful for:
- Bug reports ("here's my local state")
- Migration ("export from this browser, import in another")
- Audit ("here's exactly what this site stores")

### Clear All

One button to remove everything SG/Send has stored. A clean slate. This should warn about consequences (losing private keys if they haven't been backed up).

### Private Key Safety

Private keys are shown by fingerprint only. The actual private key material is NEVER displayed in the viewer unless the user explicitly clicks "Export" and confirms. This prevents shoulder-surfing or accidental exposure.

```
🔑 Key 1: Ed25519
   Fingerprint: b12-a7f3-c9d2-e5f1
   Created: 2026-02-15
   
   [View Public Key]  [Export Private Key ⚠️]
   
   Clicking "Export Private Key" shows a confirmation:
   "This will display your private key. Make sure nobody 
    can see your screen. Continue?"
```
