## Origin

A CISO friend (and active agent adopter) was shown the folder workflow and vaults. He described how he uses the Perplexity API for pre-meeting research on companies and individuals. This triggered the idea: stop selling files and folders. Sell specific workflows.

The question is not "how do we get people to upload files?" The question is: "what can we give an investor or a startup founder that makes them want to use this right now?"

## The Workflow: Investor Research Mode

A self-contained, multi-step research pipeline that takes a company or individual as input and produces a complete research pack in a vault.

### The Steps

```
Step 1:  Enter the target (company name, individual, or both)
Step 2:  Perplexity API researches the target (public data, news, filings, social)
Step 3:  Claude or another LLM analyses the research against criteria
         (use the published seven-point startup evaluation framework)
Step 4:  Gemini or ChatGPT generates infographics and visual summaries
Step 5:  All outputs are packaged into a vault:
         - Raw research data
         - Structured analysis report
         - Infographics and visuals
         - Source references
Step 6:  User reviews the vault contents
Step 7:  User shares the vault (download, share link, or push to recipients)
```

Every step happens in the browser. Every output is encrypted before it touches the server. The user controls what gets shared and with whom.

### What Makes This Sellable

This is not "encrypted file transfer." This is: "enter a company name, get a complete research pack in 60 seconds, encrypted and ready to share with your team."

The packaging matters. "Investor Research Mode" or "Investor Vault" is a product you can put a price on. It is a specific outcome, not a general tool.

## The Second Variant: Startup Investor Pack Creation

The same pipeline, inverted. Instead of researching a company, the startup founder packages themselves for investors.

### The Flow

```
Step 1:  Upload materials (GitHub repo, MVP link, pitch deck, vibe-coded prototype)
Step 2:  Perplexity researches the market and competitive landscape
Step 3:  LLM analyses the startup against the evaluation framework
Step 4:  LLM generates a structured pitch: problem, solution, market, traction, team
Step 5:  Gemini/ChatGPT creates visual assets (slides, infographics)
Step 6:  Everything packaged into a vault: the investor deck, the analysis,
         the market research, the visual assets, the source code review
Step 7:  Founder reviews, edits, shares with investors
```

This is the vibe coding audience. Someone who has built an MVP in a weekend and needs to turn it into something they can show investors. The workflow takes them from half-baked ideas to a complete, shareable investor pack.

## Multi-LLM Architecture

Each step uses the best model for the job:

| Step | Model | Why |
|---|---|---|
| Research | Perplexity API | Built for web research, citations, real-time data |
| Analysis | Claude | Strong at structured reasoning against frameworks |
| Infographics | Gemini or ChatGPT (image generation) | Visual output |
| Editing/refinement | Claude or user's choice | Interactive refinement |

This is the first workflow that wires multiple LLM providers together in a single pipeline. The user does not need to manage API keys for each provider. SG/Send handles the orchestration, the user pays credits.

## Revenue Model

Each research workflow consumes credits:

- Perplexity API call (research cost + markup)
- LLM analysis call (Claude/OpenRouter cost + markup)
- Image generation call (Gemini/ChatGPT cost + markup)
- Vault creation and storage (existing vault pricing)
- Sharing/downloads (existing bandwidth pricing)

A complete investor research pack might cost 2-5 GBP in credits. The value delivered (60 seconds of automated research vs. hours of manual work) justifies the price easily.
