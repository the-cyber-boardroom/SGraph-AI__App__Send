# Task: Create og:image Social Preview Image

**Date:** 2026-03-01

**Priority:** Low (enhancement)
**Status:** Open

---

## Context

Open Graph meta tags (`og:title`, `og:description`, `og:url`, `og:type`, `og:site_name`) have been added to the root `sgraph_ai__website/index.html` to fix blank social preview cards on WhatsApp, LinkedIn, and other platforms.

However, `og:image` is still missing. Without it, social previews show text-only cards. With a proper `og:image`, platforms render a visual thumbnail alongside the description — significantly increasing engagement.

## Deliverable

Create a social preview image for `sgraph.ai` (and later `send.sgraph.ai`):

- **Dimensions:** 1200 x 630 px (standard Open Graph image size)
- **Format:** PNG or JPEG (PNG preferred for crisp text/logos)
- **Content suggestions:**
  - SGraph AI logo / brand mark
  - Tagline: "AI Agents That Build, Use, and Secure the Same Platform"
  - Clean, on-brand design using the aurora theme palette
  - Legible at small thumbnail sizes (WhatsApp renders ~300px wide)
- **File location:** `sgraph_ai__website/_common/images/og-preview.png`

## Integration

Once the image is created, add this tag to the `<head>` of `sgraph_ai__website/index.html`:

```html
<meta property="og:image" content="https://sgraph.ai/_common/images/og-preview.png">
```

Also add to the `generate_i18n_pages.py` script so locale pages get the same image (with absolute URL — og:image must be a full URL, not relative).
