## Risk Reduction

| Risk Eliminated | Impact |
|---|---|
| Third-party hosting dependency | No Vercel/Netlify lock-in. No surprise pricing changes. No vendor outages affecting the marketing site. |
| Manual deployment | Fully automated. Push and it deploys. No SSH, no FTP, no console uploads. |
| Rollback complexity | IFD versioning preserves every release. Rolling back is a single S3 copy operation. |
| CDN vendor lock-in | CloudFront is current, but the deploy script targets S3. Switching CDN providers changes the distribution layer, not the deployment pipeline. |
| Supply chain exposure | No external fonts, scripts, or analytics. Zero attack surface from third-party dependencies. |
| Cost surprises | S3 + CloudFront for static sites has a predictable, near-zero cost floor. No autoscaling bills. |
