## New Pattern Introduced

### Role Activity Logs

Each role now maintains `team/roles/{role}/activity-log.md` tracking all reviews pinned to version and commit. This solves the "where did we leave off?" problem that occurs when an agent starts a new session and needs to find the latest review for a given role.

**Format:** A single markdown table with Date, Version, Commit, Review (relative link), and Description columns.

**See:** [Guide: Role Activity Log Pattern](v0.11.21__guide__role-activity-log-pattern.md) for full specification.
