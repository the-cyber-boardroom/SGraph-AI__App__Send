## Lessons Learned (For Future Sessions)

1. **CSS height propagation breaks at 3+ nesting levels.** Use JS + ResizeObserver.
2. **Component extraction is cheaper early.** Extract before the parent grows too coupled.
3. **Shell CSS should be permissive.** Don't constrain children with max-width.
4. **IFD versioning works under rapid iteration.** 18 commits, all within v0.1.6.
5. **Build → integrate → refactor is healthy.** The 3-row layout pivot mid-session improved the UX significantly.
