### Theme 1: Intelligence Tiers and Browser Automation (06 Apr — docs 236–239)

Chrome extension and Serverless Playwright API provide two complementary data capture paths (interactive vs scheduled). Vault is the universal communication layer for both.

**Roles impacted:** Dev, DevOps, Architect, QA, Conductor (revenue strategy)

**Roles impacted:** Architect (lead), Dev, AppSec

### Theme 3: UI/Tools Integration (07–08 Apr — docs 240, 242, 248)

Cross-linking tools from the main UI (doc 240) closes the island problem — powerful components exist but are unconnected. Video playback (doc 248) enables a content workflow. The folder-view vs vault explainer (doc 242) is an urgent UX/content task surfaced by a partner call.

**Roles impacted:** Dev, Designer, Ambassador
