## Milestone Event: SSH Key Generator — First Public LinkedIn Post with Video Demo

On 22 February 2026, Dinis Cruz published a LinkedIn post with a video demonstration of the browser-based SSH Key Pair Generator tool. This is the **first public marketing announcement** of the SSH key generator utility and the **second LinkedIn post** about the SG/Send project (following the PKI milestone post on 20 Feb).

### Post Content Summary

The post announces the SSH Key Pair Generator at `send.sgraph.ai/send/tools/ssh-keygen/v0/v0.1/v0.1.0/index.html` with a video showing the tool in action. Key points communicated:

- **Ed25519 (recommended) and RSA 4096** algorithm support
- **Optional comment field** for key identification
- **Everything happens in the browser** — nothing leaves the page
- **No ads, zero cookies, no tracking**
- Part of the **SGraph Send** zero-knowledge encrypted file sharing platform

- Video demo showing the SSH Key Generator tool generating key pairs in the browser

## Significance

1. **First marketing campaign activation.** The Ambassador had defined the SSH key generator as campaign A4 (priority B3/D3). This LinkedIn post is the first organic content for that campaign.
2. **Video demo format.** First time the project has published a video walkthrough — a format recommended by the Ambassador for developer tool marketing.
3. **Validates the utility-as-funnel strategy.** The SSH key generator was designed as a top-of-funnel tool to build awareness of SG/Send. This post tests whether the developer community responds to a free, privacy-respecting utility.
4. **Establishes posting cadence.** Second LinkedIn post in 3 days (after the PKI milestone post on 20 Feb). Signals regular public communication about the project.
