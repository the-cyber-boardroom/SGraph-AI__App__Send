## Key Architectural Decisions

| # | Decision | Rationale |
|---|----------|-----------|
| 1 | `_gallery.{hash}/` content-addressed folder name | Hash was already computed; enables deduplication and caching |
| 2 | Gallery folder generated only for Gallery delivery | Browse and Download must deliver unmodified user files |
| 3 | No folder view filtering | Folder view must faithfully represent zip contents |
| 4 | `SgPrint` as shared module in `v0.2.0/_common/js/` | Reusable across all UI versions via IFD symlinks |
| 5 | `@page` without `size: A4` | Chrome restricts paper size options when size is locked |
| 6 | Regenerate friendly key on Back navigation | Prevents transfer ID collision with server |
| 7 | PDF generation deferred | Requires user-controlled options; briefing document created for future session |
