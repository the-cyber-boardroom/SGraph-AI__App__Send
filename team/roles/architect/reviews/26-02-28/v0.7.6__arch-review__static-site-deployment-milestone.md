## Items Requiring Manual CloudFront Configuration

Two items need manual AWS console/CLI action before the architecture is fully complete:

1. **CloudFront Function for URL rewriting** — `url-rewrite.js` converts trailing-slash URIs to `index.html`. Without this, directory-style navigation returns S3 403 XML errors.

2. **Custom error response** — CloudFront should return the branded `404.html` page for 403/404 responses from S3 origin.

These are one-time configuration steps that cannot be automated through the current deploy pipeline.
