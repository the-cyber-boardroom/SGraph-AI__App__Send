## Tags Added Per Page

### 1. Canonical URL (all 20 pages)

```html
<link rel="canonical" href="https://sgraph.ai/en-gb/{path}">
```

Points to the definitive URL for each page. Prevents duplicate content issues across locales.

### 2. Open Graph tags (all 20 pages)

```html
<meta property="og:type" content="website">
<meta property="og:site_name" content="SGraph AI">
<meta property="og:title" content="{page title}">
<meta property="og:description" content="{page meta description}">
<meta property="og:url" content="https://sgraph.ai/en-gb/{path}">
<meta property="og:locale" content="en_GB">
```

Enables rich previews when links are shared on Facebook, LinkedIn, and other platforms.

### 3. Twitter Card tags (all 20 pages)

```html
<meta name="twitter:card" content="summary">
<meta name="twitter:title" content="{page title}">
<meta name="twitter:description" content="{page meta description}">
```

Enables summary card previews when links are shared on Twitter/X.

### 4. Hreflang alternate links (all 20 pages)

16 language-region variants plus x-default:

- `en-gb`, `en-us`
- `pt-pt`, `pt-br`
- `es-es`, `es-ar`, `es-mx`
- `fr-fr`, `fr-ca`
- `de-de`, `de-ch`
- `it-it`
- `nl-nl`
- `pl-pl`
- `ro-ro`
- `hr-hr`
- `x-default` (points to `en-gb`)

Note: `tlh` (Klingon) is intentionally excluded -- it is not a valid ISO language code and Google will reject it.

## Implementation Notes

1. **Insertion point:** All SEO meta tags were inserted immediately after the existing `<meta name="description">` tag in each file, before the `<link rel="icon">` tag.
2. **JSON-LD placement:** Structured data scripts were placed just before `</head>` on the three key pages.
3. **Existing tags preserved:** All existing `<meta>`, `<link>`, `<style>`, and `<title>` tags remain intact.
4. **Description values:** The `og:description` and `twitter:description` values match the existing `<meta name="description">` content for each page.
5. **Title values:** The `og:title` and `twitter:title` values match the existing `<title>` tag content for each page.
6. **Path correctness:** Deep pages like `pricing/free/`, `agents/ambassador/`, and `payment/cancel/` all have correct full paths in canonical URLs and hreflang links.
