# v0.7.19 Dev Review: BaseComponent Implementation Plan for v0.2.0

**Date:** 2026-03-01
**Role:** Dev
**Version:** v0.7.19
**Context:** Implementation plan for adopting MGraph's BaseComponent pattern in SGraph Send v0.2.0
**Depends on:** Architect review `v0.7.19__review__base-component-architecture-for-v0.2.0.md`

---

## 1. Executive Summary

This review provides the implementation plan for building SGraph Send's component system based on MGraph's `BaseComponent` pattern. The work produces:

1. **`SendComponent`** — base class adapted from MGraph's `BaseComponent`
2. **`SendComponentPaths`** — file path registry with IFD version awareness
3. **`SendHelpers`** — utility class with domain-specific helpers
4. **7 user UI components** — each with separate `.js`, `.html`, `.css` files
5. **Shared CSS** — `shared-components.css` + `design-tokens.css`

### Current State vs Target State

| Aspect | v0.1.x (Current) | v0.2.0 (Target) |
|---|---|---|
| Base class | None — each component extends `HTMLElement` directly | `SendComponent` — lifecycle, resource loading, utilities |
| Templates | Inline in JS (`this.innerHTML = '...'`) | Separate `.html` files fetched at runtime |
| Styles | Inline in JS or global CSS files | Separate `.css` per component, loaded into Shadow DOM |
| Shadow DOM | Mixed (admin: Shadow, user: Light) | All Shadow DOM |
| File count per component | 1 file (JS with embedded HTML+CSS) | 3 files (JS + HTML + CSS) |
| IFD override granularity | Full class redefinition (300-600 lines) | Patch individual file (HTML, CSS, or JS independently) |
| Event cleanup | Manual per-component (stored in instance vars) | Automatic via `addTrackedListener()` + `cleanup()` |
| Boilerplate per component | ~40 lines (constructor, connected, disconnected, render) | ~10 lines (just the hooks: `bindElements`, `setupEventListeners`, `onReady`) |

## 2. Implementation: SendComponent Base Class

### 2.1. Full Implementation

```javascript
/* ═══════════════════════════════════════════════════════════════════════════════
   SGraph Send — Base Component Class
   v0.2.0 — Foundation for Shadow DOM components with resource loading

   Adapted from MGraph's BaseComponent pattern.

   Provides:
   - Shadow DOM encapsulation
   - Async CSS/HTML template loading from separate files
   - Event listener tracking and cleanup
   - Common utility method access via SendHelpers
   - Standardized lifecycle hooks
   ═══════════════════════════════════════════════════════════════════════════════ */

class SendComponent extends HTMLElement {

    constructor() {
        super();
        this.attachShadow({ mode: 'open' });
        this._eventListeners  = [];
        this._isReady          = false;
        this._resourcesLoaded  = false;
    }

    // ═══════════════════════════════════════════════════════════════════════════
    // Lifecycle
    // ═══════════════════════════════════════════════════════════════════════════

    async connectedCallback() {
        try {
            await this.loadResources();
            this.bindElements();
            this.setupEventListeners();
            this._isReady = true;
            this.onReady();
            this.emit('component-ready', { component: this.tagName.toLowerCase() });
        } catch (error) {
            console.error(`[${this.tagName}] Failed to initialize:`, error);
            this.showError(`Failed to load component: ${error.message}`);
        }
    }

    disconnectedCallback() {
        this.cleanup();
    }

    /** Override in subclass for post-ready initialization */
    onReady() {}

    /** Override in subclass to bind element references after template loads */
    bindElements() {}

    /** Override in subclass to setup event listeners */
    setupEventListeners() {}

    // ═══════════════════════════════════════════════════════════════════════════
    // Resource Loading
    // ═══════════════════════════════════════════════════════════════════════════

    async loadResources() {
        if (this._resourcesLoaded) return;

        const componentName = this.tagName.toLowerCase();
        const paths         = SendComponentPaths.resolve(componentName);

        // Load CSS and HTML in parallel
        const [sharedCss, componentCss, templateHtml] = await Promise.all([
            this.fetchCss(SendComponentPaths.sharedCss.components()),
            this.fetchCss(paths.css),
            this.fetchHtml(paths.html)
        ]);

        // Inject into Shadow DOM
        this.shadowRoot.innerHTML = `
            <style>${sharedCss}</style>
            <style>${componentCss}</style>
            ${templateHtml}
        `;

        this._resourcesLoaded = true;
    }

    async fetchCss(url) {
        try {
            const response = await fetch(url);
            if (!response.ok) return '';        // No CSS — that's fine
            return await response.text();
        } catch {
            return '';                           // Network error — degrade gracefully
        }
    }

    async fetchHtml(url) {
        try {
            const response = await fetch(url);
            if (!response.ok) {
                throw new Error(`HTTP ${response.status}: ${response.statusText}`);
            }
            return await response.text();
        } catch (error) {
            console.error(`[${this.tagName}] Failed to load template: ${url}`, error);
            throw error;
        }
    }

    // ═══════════════════════════════════════════════════════════════════════════
    // Event Helpers
    // ═══════════════════════════════════════════════════════════════════════════

    emit(eventName, detail = {}) {
        this.dispatchEvent(new CustomEvent(eventName, {
            detail,
            bubbles : true,
            composed: true          // Crosses Shadow DOM boundary
        }));
    }

    addTrackedListener(target, eventType, handler, options = {}) {
        const boundHandler = handler.bind(this);
        target.addEventListener(eventType, boundHandler, options);
        this._eventListeners.push({ target, eventType, handler: boundHandler, options });
    }

    $(selector)  { return this.shadowRoot.querySelector(selector);    }
    $$(selector) { return this.shadowRoot.querySelectorAll(selector); }

    // ═══════════════════════════════════════════════════════════════════════════
    // Utility Accessors (delegate to SendHelpers)
    // ═══════════════════════════════════════════════════════════════════════════

    escapeHtml(text)     { return SendHelpers.escapeHtml(text);     }
    formatNumber(num)    { return SendHelpers.formatNumber(num);    }
    formatBytes(bytes)   { return SendHelpers.formatBytes(bytes);   }
    isValidUrl(url)      { return SendHelpers.isValidUrl(url);      }
    copyToClipboard(txt) { return SendHelpers.copyToClipboard(txt); }

    // ═══════════════════════════════════════════════════════════════════════════
    // Error Display
    // ═══════════════════════════════════════════════════════════════════════════

    showError(message) {
        this.shadowRoot.innerHTML = `
            <style>
                .component-error {
                    padding: 20px;
                    background: var(--color-error-bg, #fee2e2);
                    border: 1px solid var(--color-error-border, #fecaca);
                    border-radius: 8px;
                    color: var(--color-error, #dc2626);
                    font-family: var(--font-body, system-ui, sans-serif);
                }
                .component-error-title {
                    font-weight: 600;
                    margin-bottom: 8px;
                }
            </style>
            <div class="component-error">
                <div class="component-error-title">Component Error</div>
                <div>${this.escapeHtml(message)}</div>
            </div>
        `;
    }

    // ═══════════════════════════════════════════════════════════════════════════
    // Cleanup
    // ═══════════════════════════════════════════════════════════════════════════

    cleanup() {
        for (const { target, eventType, handler, options } of this._eventListeners) {
            target.removeEventListener(eventType, handler, options);
        }
        this._eventListeners = [];
        this._isReady = false;
    }

    // ═══════════════════════════════════════════════════════════════════════════
    // Ready State
    // ═══════════════════════════════════════════════════════════════════════════

    get isReady() { return this._isReady; }

    whenReady(timeout = 5000) {
        if (this._isReady) return Promise.resolve();
        return new Promise((resolve, reject) => {
            const timer = setTimeout(() => {
                reject(new Error(`Component ${this.tagName} ready timeout`));
            }, timeout);
            this.addEventListener('component-ready', () => {
                clearTimeout(timer);
                resolve();
            }, { once: true });
        });
    }
}
```

**File location:** `_common/js/base/send-component.js`
**Lines:** ~130 (vs MGraph's ~180 — trimmed Node.js export and global assignment)

### 2.2. Key Adaptations from MGraph

| MGraph | SGraph Send | Why |
|---|---|---|
| `window.BaseComponent = BaseComponent` | No global assignment | Components load after base class — no timing issue |
| `module.exports = BaseComponent` | Removed | Browser-only, no Node.js |
| Error uses emoji `⚠️` | Error uses text "Component Error" | CLAUDE.md rule: no emojis unless requested |
| Hardcoded error colors | Uses CSS custom properties | Respects design tokens from `design-tokens.css` |

## 3. Implementation: SendComponentPaths

```javascript
/* ═══════════════════════════════════════════════════════════════════════════════
   SGraph Send — Component Path Registry
   v0.2.0 — Maps component tag names to file paths
   ═══════════════════════════════════════════════════════════════════════════════ */

class SendComponentPaths {

    // Set at boot time based on page location
    // From en-gb/index.html: basePath = '../_common'
    // From en-gb/upload/index.html: basePath = '../../_common'
    static basePath = '../_common';

    static sharedCss = {
        components: () => `${SendComponentPaths.basePath}/css/shared-components.css`
    };

    static resolve(tagName) {
        const base = `${SendComponentPaths.basePath}/js/components/${tagName}/${tagName}`;
        return {
            js:   `${base}.js`,
            html: `${base}.html`,
            css:  `${base}.css`
        };
    }

    /**
     * Set base path from current page depth.
     * Call this before any component is defined.
     *
     * @param {string} path - Relative path to _common/ from current HTML page
     */
    static init(path) {
        SendComponentPaths.basePath = path;
    }
}
```

**File location:** `_common/js/base/send-component-paths.js`

### 3.1. Usage in Locale Pages

```html
<!-- en-gb/index.html (1 level deep from version root) -->
<script src="../_common/js/base/send-component-paths.js"></script>
<script>SendComponentPaths.init('../_common');</script>

<!-- en-gb/upload/index.html (2 levels deep) -->
<script src="../../_common/js/base/send-component-paths.js"></script>
<script>SendComponentPaths.init('../../_common');</script>
```

### 3.2. IFD Override Pattern (v0.2.1 Example)

```javascript
/* v0.2.1 override — only send-upload template changed */
(function() {
    const originalResolve = SendComponentPaths.resolve;

    SendComponentPaths.resolve = function(tagName) {
        const paths = originalResolve.call(this, tagName);

        // Override specific files for specific components
        if (tagName === 'send-upload') {
            paths.html = '../../v0.2.1/_common/js/components/send-upload/send-upload.html';
        }

        return paths;
    };
})();
```

This is a **10-line file** that redirects just one component's template to the new version. All other components continue loading from v0.2.0.

## 4. Implementation: SendHelpers

```javascript
/* ═══════════════════════════════════════════════════════════════════════════════
   SGraph Send — Utility Helpers
   v0.2.0 — Shared formatting, validation, and display utilities
   ═══════════════════════════════════════════════════════════════════════════════ */

const SendHelpers = {

    // ─── Text Safety ──────────────────────────────────────────────────────

    escapeHtml(text) {
        const div = document.createElement('div');
        div.textContent = text;
        return div.innerHTML;
    },

    // ─── Number Formatting ────────────────────────────────────────────────

    formatNumber(num) {
        return new Intl.NumberFormat().format(num);
    },

    formatBytes(bytes) {
        if (bytes === 0) return '0 B';
        const units = ['B', 'KB', 'MB', 'GB', 'TB'];
        const i = Math.floor(Math.log(bytes) / Math.log(1024));
        const value = bytes / Math.pow(1024, i);
        return `${value.toFixed(i === 0 ? 0 : 1)} ${units[i]}`;
    },

    // ─── URL Validation ──────────────────────────────────────────────────

    isValidUrl(str) {
        try {
            new URL(str);
            return true;
        } catch {
            return false;
        }
    },

    // ─── Clipboard ───────────────────────────────────────────────────────

    async copyToClipboard(text) {
        try {
            await navigator.clipboard.writeText(text);
            return true;
        } catch {
            // Fallback for older browsers
            const textarea = document.createElement('textarea');
            textarea.value = text;
            textarea.style.position = 'fixed';
            textarea.style.opacity = '0';
            document.body.appendChild(textarea);
            textarea.select();
            const ok = document.execCommand('copy');
            document.body.removeChild(textarea);
            return ok;
        }
    },

    // ─── Event Throttling ────────────────────────────────────────────────

    debounce(fn, wait) {
        let timer;
        return function(...args) {
            clearTimeout(timer);
            timer = setTimeout(() => fn.apply(this, args), wait);
        };
    },

    // ─── SGraph Send Domain Helpers ──────────────────────────────────────

    /** Format a 12-char transfer ID for display: "abc1…f456" */
    formatTransferId(id) {
        if (!id || id.length < 8) return id || '';
        return `${id.slice(0, 4)}…${id.slice(-4)}`;
    },

    /** Format a base64 encryption key for display (truncated) */
    formatKey(key) {
        if (!key) return '';
        return key.length > 24 ? `${key.slice(0, 24)}…` : key;
    },

    /** Validate a transfer ID (12 alphanumeric chars) */
    validateTransferId(id) {
        return /^[a-zA-Z0-9_-]{12}$/.test(id);
    },

    /** Build a locale-aware download URL */
    buildDownloadUrl(transferId, key, locale) {
        const loc = locale || 'en-gb';
        const fragment = key ? `${transferId}/${key}` : transferId;
        return `${window.location.origin}/${loc}/download/#${fragment}`;
    }
};
```

**File location:** `_common/js/base/send-helpers.js`

## 5. Implementation: Example Component — SendUpload

### 5.1. Component JS — `send-upload.js`

```javascript
/* ═══════════════════════════════════════════════════════════════════════════════
   SGraph Send — Upload Component
   v0.2.0 — Consolidated from v0.1.0 through v0.1.8

   Extends SendComponent (MGraph BaseComponent pattern).
   Template and styles loaded from separate files.
   ═══════════════════════════════════════════════════════════════════════════════ */

class SendUpload extends SendComponent {

    constructor() {
        super();
        this.selectedFile = null;
        this.state = 'idle';   // idle | encrypting | uploading | complete | error
    }

    // ─── Lifecycle Hooks ──────────────────────────────────────────────────

    bindElements() {
        this._dropZone   = this.$('#drop-zone');
        this._fileInput  = this.$('#file-input');
        this._uploadBtn  = this.$('#upload-btn');
        this._progress   = this.$('#progress');
        this._result     = this.$('#result');
        this._errorPanel = this.$('#error-panel');
        this._fileInfo   = this.$('#file-info');
    }

    setupEventListeners() {
        if (this._dropZone) {
            this.addTrackedListener(this._dropZone, 'dragover',  this.onDragOver);
            this.addTrackedListener(this._dropZone, 'dragleave', this.onDragLeave);
            this.addTrackedListener(this._dropZone, 'drop',      this.onDrop);
            this.addTrackedListener(this._dropZone, 'click',     this.onDropZoneClick);
        }
        if (this._fileInput) {
            this.addTrackedListener(this._fileInput, 'change', this.onFileSelected);
        }
        if (this._uploadBtn) {
            this.addTrackedListener(this._uploadBtn, 'click', this.startUpload);
        }
    }

    onReady() {
        // Translate any data-i18n elements in the shadow DOM
        if (typeof I18n !== 'undefined') {
            this.$$('[data-i18n]').forEach(el => {
                el.textContent = I18n.t(el.getAttribute('data-i18n'));
            });
        }
    }

    // ─── Drag & Drop ─────────────────────────────────────────────────────

    onDragOver(e) {
        e.preventDefault();
        this._dropZone.classList.add('drop-zone--active');
    }

    onDragLeave(e) {
        e.preventDefault();
        this._dropZone.classList.remove('drop-zone--active');
    }

    onDrop(e) {
        e.preventDefault();
        this._dropZone.classList.remove('drop-zone--active');
        if (e.dataTransfer.files.length > 0) {
            this.selectFile(e.dataTransfer.files[0]);
        }
    }

    onDropZoneClick() {
        this._fileInput.click();
    }

    onFileSelected(e) {
        if (e.target.files.length > 0) {
            this.selectFile(e.target.files[0]);
        }
    }

    // ─── File Selection ──────────────────────────────────────────────────

    selectFile(file) {
        this.selectedFile = file;
        this._fileInfo.innerHTML = `
            <span><strong>${this.escapeHtml(file.name)}</strong> (${this.formatBytes(file.size)})</span>
        `;
        this._fileInfo.hidden = false;
        this._uploadBtn.hidden = false;
        this._uploadBtn.disabled = false;
    }

    // ─── Upload Flow ─────────────────────────────────────────────────────

    async startUpload() {
        if (!this.selectedFile || this.state !== 'idle') return;

        try {
            // 1. Generate key
            this.setState('encrypting');
            const key = await SendCrypto.generateKey();
            const keyString = await SendCrypto.exportKey(key);

            // 2. Encrypt
            const encrypted = await SendCrypto.encrypt(key, await this.selectedFile.arrayBuffer());

            // 3. Create transfer
            this.setState('uploading');
            const createResult = await ApiClient.createTransfer({
                file_size: encrypted.byteLength
            });

            // 4. Upload encrypted blob
            await ApiClient.uploadBlob(createResult.transfer_id, encrypted);

            // 5. Complete transfer
            await ApiClient.completeTransfer(createResult.transfer_id);

            // 6. Show result
            const downloadUrl = SendHelpers.buildDownloadUrl(
                createResult.transfer_id,
                keyString,
                document.documentElement.lang
            );

            this.setState('complete');
            this.showResult(createResult.transfer_id, downloadUrl, keyString);

            // 7. Emit event
            this.emit('upload-complete', {
                transferId:  createResult.transfer_id,
                downloadUrl: downloadUrl,
                key:         keyString
            });

        } catch (error) {
            console.error('[send-upload] Upload failed:', error);
            this.setState('error');
            this.showUploadError(error.message);
        }
    }

    // ─── State Management ────────────────────────────────────────────────

    setState(newState) {
        this.state = newState;
        // Toggle visibility of sections
        if (this._dropZone)   this._dropZone.hidden   = (newState === 'complete');
        if (this._progress)   this._progress.hidden   = !(newState === 'encrypting' || newState === 'uploading');
        if (this._result)     this._result.hidden     = (newState !== 'complete');
        if (this._errorPanel) this._errorPanel.hidden = (newState !== 'error');
        if (this._uploadBtn)  this._uploadBtn.disabled = (newState !== 'idle');

        // Update progress text
        if (this._progress && !this._progress.hidden) {
            const label = this._progress.querySelector('.progress__label');
            if (label) {
                label.textContent = newState === 'encrypting' ? 'Encrypting...' : 'Uploading...';
            }
        }
    }

    showResult(transferId, downloadUrl, key) {
        if (!this._result) return;
        this._result.innerHTML = `
            <div class="result__section">
                <label class="result__label" data-i18n="result.download_link">Download Link</label>
                <div class="result__value">
                    <code class="result__code">${this.escapeHtml(downloadUrl)}</code>
                    <button class="btn btn-sm btn-secondary result__copy" data-copy="${this.escapeHtml(downloadUrl)}">Copy</button>
                </div>
            </div>
            <div class="result__section">
                <label class="result__label" data-i18n="result.decryption_key">Decryption Key</label>
                <div class="result__value">
                    <code class="result__code">${this.escapeHtml(key)}</code>
                    <button class="btn btn-sm btn-secondary result__copy" data-copy="${this.escapeHtml(key)}">Copy</button>
                </div>
            </div>
        `;
        this._result.hidden = false;

        // Bind copy buttons
        this._result.querySelectorAll('.result__copy').forEach(btn => {
            this.addTrackedListener(btn, 'click', () => {
                this.copyToClipboard(btn.dataset.copy);
                btn.textContent = 'Copied!';
                setTimeout(() => btn.textContent = 'Copy', 2000);
            });
        });

        // Translate
        if (typeof I18n !== 'undefined') {
            this._result.querySelectorAll('[data-i18n]').forEach(el => {
                el.textContent = I18n.t(el.getAttribute('data-i18n'));
            });
        }
    }

    showUploadError(message) {
        if (!this._errorPanel) return;
        this._errorPanel.innerHTML = `
            <div class="status status--error">${this.escapeHtml(message)}</div>
        `;
        this._errorPanel.hidden = false;
    }
}

customElements.define('send-upload', SendUpload);
```

**Key design decisions:**
- Pure HTML — no `<script>`, no `<style>`
- All text has `data-i18n` attributes for translation
- IDs match what `bindElements()` queries
- Hidden sections toggled by JS state machine
- SVG inline for the drop zone icon (no external asset dependency)

## 8. Migration: v0.1.x → v0.2.0 Component by Component

### 8.1. Migration Steps Per Component

For each of the 7 components:

1. **Extract HTML template** from inline `this.innerHTML` / `this.shadowRoot.innerHTML` → separate `.html` file
2. **Extract CSS** from inline `<style>` blocks or global CSS → separate `.css` file
3. **Refactor JS** to extend `SendComponent` instead of `HTMLElement`
   - Remove `constructor() { super(); this.attachShadow(...) }` → inherited
   - Remove manual event listener storage → use `addTrackedListener()`
   - Remove `render()` method → template comes from `.html` file
   - Add `bindElements()` → query Shadow DOM elements by ID
   - Add `setupEventListeners()` → bind events to elements
   - Add `onReady()` → post-template initialization
4. **Test** — verify component renders, events work, i18n translates

### 8.2. Migration Priority Order

|---|---|---|
| 1 | `send-access-gate` | Simplest — gate logic only, no crypto |

```javascript
// v0.2.1 surgical patch — 20 lines
(function() {
    const originalUpload = SendUpload.prototype.startUpload;

    SendUpload.prototype.startUpload = async function() {
        const maxRetries = 3;
        for (let attempt = 1; attempt <= maxRetries; attempt++) {
            try {
                return await originalUpload.call(this);
            } catch (error) {
                if (attempt === maxRetries) throw error;
                await new Promise(r => setTimeout(r, 1000 * attempt));
            }
        }
    };
})();
```

**Total new code: 20 lines**
