## Design Decisions

### Asset paths (from `en-gb/*.html`)
- Fonts: `../../../../_common/fonts/fonts.css` (shared, existing)
- CSS: `../_common/css/style.css` + page CSS (within v0.2.0)
- JS: `../_common/js/*.js` (within v0.2.0)
- Favicon: `../../../../_common/favicon.svg` (shared, existing)

### No inline styles or scripts
All CSS is in `_common/css/`.
All JS is in `_common/js/`.
HTML files contain only structure and content.

### Screenshot placeholders
Use `.screenshot-placeholder` component (teal dashed border, dark background).
Drop in `<img>` tags when screenshots arrive — no layout changes needed.

### Nav structure (v0.2.0)
`How it Works | Security | Tools | Pricing` — replaces `Product | Pricing | Architecture | Team`
Persistent `[Paste a token →]` button expands inline input in nav bar.
