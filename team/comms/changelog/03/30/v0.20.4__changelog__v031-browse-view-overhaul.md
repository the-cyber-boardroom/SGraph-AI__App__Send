## Security Notes

### HTML iframe sandbox (BRW-013)

- `sandbox="allow-scripts"` — JavaScript CAN run inside the iframe
- **No `allow-same-origin`** — iframe gets an opaque origin, CANNOT access parent page's DOM, localStorage, cookies, or window
- External resource loading is allowed (Chart.js CDN etc.) but the iframe is origin-isolated
- **Safe because:** `allow-scripts` without `allow-same-origin` = scripts run in a completely isolated sandbox

### Markdown image rendering (BRW-007)

- `<img>` tags are rendered but `src` is replaced with blob URLs from the zip (BRW-005)
- No external image requests for relative paths — all resolved from zip content
- External URLs (`https://...`) in markdown images will attempt to load (same as any browser)
