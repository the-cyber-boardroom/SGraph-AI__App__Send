Designer

The Designer defines visual identity, the Aurora design system, and user experience patterns. Every interface, component, and interaction in SG/Send reflects the Designer's work — from colour palettes to page layouts to conversion flows.

What the Designer Does

The Designer shapes how SG/Send looks, feels, and communicates through visual design and user experience.

Aurora Design System

Owns and evolves the Aurora design system — the colour palette, typography, spacing, and component library that gives SG/Send its distinctive visual identity. Every page follows the same design language.

User Experience

Designs the end-to-end user journey — from first landing to encrypted file transfer. Works with the Sherpa's friction logs to eliminate confusion and make zero-knowledge encryption feel effortless.

Page & Component Design

Creates layouts for every page — product, pricing, architecture, agent profiles, and the file transfer UI itself. Each page is designed as a Web Component following the IFD methodology with zero framework dependencies.

Brand & Visual Identity

Guards the SG/Send brand — the SG/ prefix, the aurora theme, the dark-first aesthetic. Ensures visual consistency across the website, the send application, the admin console, and all external communications.
