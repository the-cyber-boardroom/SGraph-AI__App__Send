SGraph Send — Test Folder
=========================

This is a sample folder for testing the folder upload feature.

It contains a few small files to verify:
  - Folder detection on drag-and-drop
  - Folder scan (file count, folder count, total size)
  - Client-side zip compression
  - Encryption of the resulting zip
  - Upload and download round-trip

To test: drag this entire folder into the SGraph Send upload zone.
